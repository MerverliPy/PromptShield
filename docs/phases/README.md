# Phases Roadmap

`docs/phases/README.md` is roadmap context only.

Active workflow state does not live in this directory. Use `AGENTS.md` for the workflow contract and `.opencode/plans/current-phase.md` for the current phase.

## Usage

- Keep roadmap notes here only when they help explain delivery order.
- Do not store active execution state, handoff notes, or validation state here.
- Use Git history for prior phase history.
